# My Writing Notes 快速入门指南

## 🚀 5 分钟快速开始

### 步骤 1：触发技能

在 Qoder 中输入：

```
"帮我写一篇关于 Sentinel 的技术笔记"
```

或

```
"Create a technical note about Nacos"
```

### 步骤 2：自动生成初稿（第一阶段）

系统自动执行：
- ✅ deep-research 深度调研
- ✅ 内容整理和写作
- ✅ 智能配图（draw.io + Mermaid + AI 提示词）
- ✅ Review 内容检查

**输出**：初稿文档（含图像提示词）

### 步骤 3：审阅初稿

检查内容：
- [ ] 技术准确性
- [ ] 代码可运行性
- [ ] 场景贴切度
- [ ] 提示词精确性

### 步骤 4：确认清理（第二阶段）

如果初稿无误，输入：

```
"初稿无误，开始清理提示词"
```

系统自动执行：
- ✅ 删除图像提示词
- ✅ 创建备份文件
- ✅ 输出纯净文档

**输出**：纯净文档 + 备份文件

## 📋 完整使用流程

### 场景 1：生成技术笔记

```
用户："帮我写一篇关于 Sentinel 流量防卫兵的技术笔记，以电商下单为场景"

系统执行：
  第一阶段（自动）：
    Phase 1: deep-research 调研 Sentinel
    Phase 2: 整理为电商场景结构
    Phase 3: 按五步讲解法写作
    Phase 4: 生成插图和提示词
    Review: 内容检查
    ↓
  输出初稿
    
  用户审阅：✅ 内容准确
  
  第二阶段（确认后）：
    Phase 5: 删除提示词
    Phase 6: 创建备份
    ↓
  输出纯净文档
```

### 场景 2：修改初稿

```
用户："帮我写一篇关于 HSF 的技术笔记"

系统生成初稿
  ↓
用户审阅：❌ 代码示例有问题
  ↓
用户："代码示例需要修改，重新生成 Phase 3"
  ↓
系统返回 Phase 3 重新写作
  ↓
输出修改后的初稿
```

### 场景 3：保留提示词

```
用户："帮我写一篇关于 TDDL 的技术笔记"

系统生成初稿
  ↓
用户审阅：✅ 但我想保留提示词，稍后手动生成图片
  ↓
用户："保留提示词，不清理"
  ↓
系统跳过第二阶段
  ↓
输出含提示词的初稿
```

## 🎨 图像生成说明

### 三种图像方式

| 方式 | 适用场景 | 示例 |
|------|---------|------|
| **draw.io** | 复杂架构图（>10 组件） | 分层架构、数据流向 |
| **Mermaid** | 简单流程图、时序图 | 调用流程、生命周期 |
| **AI 提示词** | 概念插图、场景示意 | 限流场景、降级流程 |

### AI 提示词格式

初稿中会包含：

```markdown
[🎨 生图提示词]
```
极简科技教育图解，扁平矢量插画...
```

> 📷 Sentinel 分层防护架构
```

清理后仅保留：

```markdown
> 📷 Sentinel 分层防护架构
```

## ✅ 质量检查

### Review 检查项

第一阶段完成后，系统自动检查：

- ✅ **内容逻辑一致性**
  - 章节间有因果递进
  - 场景贯穿全文
  - 技术引入自然

- ✅ **技术准确性**
  - 定义准确
  - 代码正确
  - 配置可行

- ✅ **场景连贯性**
  - 场景贴切
  - 示例符合场景
  - 类比恰当

- ✅ **代码示例完整性**
  - 代码可运行
  - 包含注释
  - 有完整上下文

### 用户检查项

第二阶段前，用户需确认：

- [ ] 技术准确性
- [ ] 代码可运行性
- [ ] 场景贴切度
- [ ] 提示词精确性
- [ ] 图片描述简洁

## 🔒 安全机制

### 数据保护

- ✅ 必须先创建备份文件
- ✅ 备份文件验证完整性
- ✅ Human 确认后才能删除
- ✅ 删除前二次确认
- ✅ 备份文件永久保留

### 删除范围

仅删除：
- `[🎨 生图提示词]` 标记
- 提示词代码块内容

保留：
- `> 📷 图片描述`
- 图像引用
- 所有正文内容

## 💡 常见问题

### Q1: 如何触发技能？

**A**: 使用触发词，如：
```
"帮我写一篇关于 X 的技术笔记"
```

### Q2: 第一阶段输出什么？

**A**: 初稿文档，包含：
- 完整正文
- draw.io 图片引用
- Mermaid 代码块
- AI 图像提示词（初稿标记）

### Q3: 如何开始第二阶段？

**A**: 审阅初稿后，输入：
```
"初稿无误，开始清理提示词"
```

### Q4: 提示词删除后能恢复吗？

**A**: 可以！备份文件永久保留：
```
resources/images/{文档名}/{文档名}-image-prompts.md
```

### Q5: 可以跳过第二阶段吗？

**A**: 可以！输入：
```
"保留提示词，不清理"
```

### Q6: 如何自定义场景？

**A**: 在请求中指定：
```
"以电商下单为场景，写 Sentinel 的技术笔记"
```
